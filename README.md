# Democracy in the 21st Century

An interactive web project exploring the development of democracy around the world over the last century. The website presents visualizations and short analyses about democratic trends, population living under democracies vs. autocracies, and the relationship between democracy and quality of life.

## Features

- Global overview of democratic development
- Historical comparison of democracy across continents
- Analysis of democracy vs. autocracy trends
- Visual exploration of the 10 most populated countries
- Correlation between democracy and happiness indicators

## Technologies

- HTML
- CSS
- Data visualizations and static graphics

## Live Website

The project is hosted at:

https://lehmannfabian.github.io/democracy_21st/

## Project Goal

The aim of this project is to provide an accessible and visual introduction to the current global democratic landscape and its historical evolution.
